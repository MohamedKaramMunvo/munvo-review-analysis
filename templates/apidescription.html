Review analysis REST API

YourAI REST API allows developers to easily integrate reviews analysis capabilities into their existing platforms. With just a simple, fast and well-built API call, you can send a review text and receive its sentiment analysis, or summarize it into one single sentence. Whether you're building a customer feedback platform, conducting market research, or monitoring social media sentiment, this API can be the perfect solution for you.

POST    https://app.youraiplatform.com/api/sentiment
POST    https://app.youraiplatform.com/api/summary
Description & request/response example
Description
Send a review text, and get a binary classification of positive or negative sentiment, along with a corresponding accuracy score.

Http Request Body
{
  "email" : "myemail@domain.com",
  "password" : "mypassword123",
  "text" : "i was very satisfied with this service"
}

Http Response

Media type : application/json
Http code   : 200 OK

{
  "sentiment": "Positive",
  "score": 98
}
Description & request/response example
Description
Send a review text, and get a concise summary that retains the same meaning as the original review, in one sentence of few words. it helps to have a summary of large texts

Http Request Body
{
  "email" : "myemail@domain.com",
  "password" : "mypassword123",
  "text" : "Product arrived labeled as Jumbo Salted Peanuts...the peanuts were
     actually small sized and unsalted. not just that but i also got a different
     box color than the one stated on the website"
}

Http Response

Media type : application/json
Http code   : 200 OK

{
  "summary": "The product was mislabeled and contained small, unsalted peanuts in
     a different box color than advertised.",
}
